# Google Drops Gemini 3.0 Flash While Everyone Was Watching Meta

**February 26, 2026**

While everyone was dissecting Meta's Llama 4 Scout announcement from this morning, Google quietly dropped **Gemini 3.0 Flash** -- and it's fast enough to make real-time agents actually real-time.

Today: Gemini 3.0 Flash benchmarks, Llama 4 Scout goes open-weight, and Cursor's monster fundraise.

---

## 🧠 MODEL RELEASES

**Gemini 3.0 Flash lands with 2M token context.** Google's latest speed demon hits 92.3% on MMLU-Pro while running 3x faster than 2.5 Flash. The 2M context window is GA -- no waitlist, no beta tag. Pricing: $0.15/M input tokens makes it the cheapest frontier model per token. [Read more →](https://blog.google/technology/ai/gemini-3-flash/)

**Meta open-sources Llama 4 Scout (17B active / 109B total).** The first MoE model in the Llama family activates 17B of its 109B parameters per forward pass. Beats Gemma 3 and GPT-4o mini on reasoning benchmarks while fitting on a single H100. Open weights, no strings. [Read more →](https://ai.meta.com/blog/llama-4-scout/)

## 🔧 DEVELOPER & PLATFORM

**Cursor raises $900M at $9B valuation**: The AI code editor that devs actually use just became the most valuable dev tool startup. CEO Michael Truell says the money goes to "agentic coding infrastructure." [Read more →](https://www.cursor.com/blog/series-c)

**OpenAI WebSocket Realtime API goes GA**: No more REST polling for real-time apps. The new WebSocket endpoint supports streaming function calls, voice, and vision in a single persistent connection. Latency drops from ~800ms to ~120ms for multi-turn agent loops. [Read more →](https://openai.com/index/realtime-api-websocket-ga/)

**GitHub Copilot Workspace adds multi-repo support**: Your AI dev environment can now reason across up to 5 linked repositories simultaneously. Finally useful for microservice architectures. [Read more →](https://github.blog/changelog/copilot-workspace-multi-repo/)

## 📝 RESEARCH & INSIGHTS

**Scaling test-time compute is more efficient than you think**: DeepMind paper shows that spending 4x on inference-time search beats spending 16x on pre-training compute for math and code tasks. The "think harder, not bigger" era is here. [Read more →](https://arxiv.org/abs/2602.12345)
