## 🎓 MODEL LITERACY

**FP4 Quantization**: When NVIDIA publishes a "FP4" version of Qwen3.5-397B, they're compressing the model's weights from 16-bit or 8-bit floating point down to just 4 bits per parameter. This means each weight uses 4x less memory than FP16, so a model that normally needs 8 GPUs might fit on 2. The tradeoff is precision — some quality is lost — but modern quantization techniques (like GPTQ and AWQ) have gotten remarkably good at preserving model capability. For practitioners, quantization is the bridge between "this model exists" and "I can actually run this model."

**The distillation scandal changes the economics of closed AI.** Anthropic catching DeepSeek, Moonshot AI, and MiniMax running 24,000 fake accounts to systematically extract Claude's capabilities isn't just a security story — it's an existential question for the closed-model business.
