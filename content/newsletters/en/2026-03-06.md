## 🎓 MODEL LITERACY

**Mixture of Experts (MoE)**: Not every parameter in a model needs to fire for every token. MoE architectures split the model into specialized "expert" sub-networks and use a router to activate only a few of them per input — Qwen 3.5's 35B-A3B model has 35 billion total parameters but only activates 3 billion at inference time. This means you get large-model quality at small-model speed and cost. The tradeoff: MoE models need more total memory to load (all experts must be in RAM), but they're dramatically cheaper to run once loaded.
