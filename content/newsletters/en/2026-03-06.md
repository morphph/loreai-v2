## 🎓 MODEL LITERACY

**Model Distillation**: You'll see this term a lot today. Distillation is when a smaller "student" model learns to mimic a larger "teacher" model by training on the teacher's outputs rather than raw data. The student can capture much of the teacher's capability at a fraction of the compute cost. The controversy: when someone distills *your* model via API calls without permission, they're essentially extracting your R&D investment. Anthropic's disclosure today shows this happening at industrial scale — 16M+ API exchanges designed to systematically copy model behavior. It's why API terms of service prohibit using outputs to train competing models, and why detection systems for this are becoming critical infrastructure.

## 🎯 PICK OF THE DAY

**The prompt injection attack that hit 4,000 machines is the wake-up call we needed.** A malicious GitHub issue title — just the *title* — was enough to hijack AI coding tools and compromise thousands of developer machines. The attack vector is elegant and terrifying: AI coding assistants that ingest repository metadata trusted that issue titles were benign text. They weren't. This is the first large-scale, real-world prompt injection supply chain attack, and it validates every security researcher who's been warning about this exact scenario for two years. The uncomfortable truth: every AI coding tool that reads untrusted input — Copilot, Cursor, Cline, Claude Code — has some version of this attack surface. The question isn't whether your tool is vulnerable, it's whether it has adequate sandboxing and input sanitization. If you're a team lead, audit what your AI tools can access and execute. Today.
