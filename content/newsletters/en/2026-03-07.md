## 🎓 MODEL LITERACY

**Mixture of Experts (MoE)**: When you see a model described as "35B total, 3B active" (like Qwen3.5-35B-A3B), that's MoE architecture at work. Instead of running every parameter for every token, the model routes each input to a small subset of specialized "expert" sub-networks. You get the knowledge capacity of a 35B model with the inference cost of a 3B model. The trade-off is memory — you still need to load all 35B parameters into VRAM, even though only 3B fire at any given time. This is why MoE models are the sweet spot for running capable models on consumer GPUs: fast inference, smart routing, big knowledge.
