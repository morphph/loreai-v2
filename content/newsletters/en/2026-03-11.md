## 🎓 MODEL LITERACY

**Mixture of Experts (MoE) — Why Qwen's 397B Model Only Uses 17B Parameters**: MoE architectures split a model into many "expert" sub-networks but only activate a small fraction of them for any given input. Qwen3.5-397B-A17B has 397 billion total parameters but routes each token through just 17 billion active ones — giving you big-model quality at small-model inference cost. The tradeoff: MoE models need more total memory to store all experts, even though compute per token stays low. This is why you're seeing huge parameter counts that can still run on reasonable hardware.
