## 🎓 MODEL LITERACY

**Mixture of Experts (MoE)**: When you see "35B total, 3B active" on a model like Qwen3.5-35B-A3B, that's MoE at work. Instead of running every parameter on every token, the model routes each input to a small subset of specialized "expert" sub-networks. You get the knowledge capacity of a 35B model but only pay the compute cost of a 3B model per inference. The tradeoff: MoE models need more memory to store all experts, but they're dramatically cheaper to actually run — which is why they're dominating the self-hosted and cost-sensitive deployment space.
