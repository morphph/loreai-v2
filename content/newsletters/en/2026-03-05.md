## 🎓 MODEL LITERACY

**Mixture of Experts (MoE)**: MoE is an architecture where a model has many parameters but only activates a subset of them for each input — like a hospital with 50 specialists but only 3 see any given patient. Qwen3.5-35B-A3B uses this: 35 billion total parameters, but only 3 billion are "active" per token, making inference dramatically cheaper and faster. The trade-off is more complex routing logic and larger model files on disk, but for local and cost-sensitive deployments, MoE is becoming the dominant strategy for punching above your compute weight class.
