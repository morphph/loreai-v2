# Google Drops Five Products in a Day While Claude Takes Over the App Store

**March 5, 2026**

Google went on a shipping spree — **Gemini 3.1 Pro**, **Gemini 3 Deep Think**, **Nano Banana 2** image gen, and music creation all landed in the same news cycle. Meanwhile, **Claude** hit #1 in the App Store and SemiAnalysis claims 4% of GitHub public commits now come from Claude Code. The machines are busy.

Today: Google's model blitz, OpenAI's record $110B raise, and Claude Code's new skills.

## 🧠 MODEL RELEASES

**Gemini 3.1 Pro lands.** Google's latest flagship is built for complex, multi-step reasoning — designed for tasks where a simple answer isn't enough. The first 3.1-series model signals Google's push toward deeper problem-solving. [Read more →](https://deepmind.google/blog/gemini-3-1-pro-a-smarter-model-for-your-most-complex-tasks/)

**Gemini 3 Deep Think gets a major update.** Google's most specialized reasoning mode now targets modern science, research, and engineering challenges. If you're doing serious technical work, this is the one to watch. [Read more →](https://deepmind.google/blog/gemini-3-deep-think-advancing-science-research-and-engineering/)

**Nano Banana 2 ships image gen at Flash speed.** Google's latest image generation model combines Pro-level world knowledge and subject consistency with lightning-fast inference — production-ready specs out of the box. [Read more →](https://deepmind.google/blog/nano-banana-2-combining-pro-capabilities-with-lightning-fast-speed/)

**Qwen3.5-35B-A3B drops on HuggingFace.** Alibaba's new multimodal model packs image-text capabilities into a 35B parameter frame with only 3B active — already pulling 680K+ downloads. Efficient and capable. [Read more →](https://huggingface.co/Qwen/Qwen3.5-35B-A3B)
