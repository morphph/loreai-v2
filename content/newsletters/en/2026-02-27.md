# Anthropic Ships Claude Code Remote While Qwen 3 Drops a Model Bomb

**February 27, 2026**

Anthropic just made Claude Code work from anywhere -- literally. **Claude Code Remote** lets you spin up cloud dev environments with full agentic capabilities, and Alibaba's **Qwen 3** dropped an entire family of 8 models overnight.

Today: Claude Code Remote, the Qwen 3 lineup, and NVIDIA's GB300 inference chip.

## 🧠 MODEL RELEASES

**Qwen 3 drops 8 models in one day.** Alibaba's AI lab went nuclear: dense models at 0.6B, 1.7B, 4B, 8B, 32B, and 72B plus two MoE variants (30B-A3B, 235B-A22B). The 72B beats Llama 4 Scout on MATH-500 and LiveCodeBench. All Apache 2.0. [Read more →](https://qwenlm.github.io/blog/qwen3/)

**Cohere Command R+ 2 ships with 256K context.** The enterprise-focused model now handles book-length documents natively. RAG accuracy up 18% on internal benchmarks vs Command R+. Priced aggressively at $2.50/M input tokens for enterprise. [Read more →](https://cohere.com/blog/command-r-plus-2)

**Vercel AI SDK 5.0 ships with agent primitives**: New `createAgent()` and `createToolkit()` APIs make building multi-step AI agents as easy as writing a React component. Built-in streaming, tool calling, and memory management. [Read more →](https://vercel.com/blog/ai-sdk-5)

**Cloudflare Workers AI adds streaming function calling**: Run AI inference at the edge with real-time tool use. Sub-200ms latency for simple agentic tasks from 300+ global PoPs. [Read more →](https://blog.cloudflare.com/workers-ai-streaming-function-calling)
