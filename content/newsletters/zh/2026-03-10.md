# Introducing Code Review, a new feature for Claude Code.  Whe，意味着什么

## 🧠 发布动态

### Claude Code 上线代码审查，PR 一开就派 Agent 大军找 Bug。
不是简单的 lint 检查 — PR 打开后，Claude 会自动调度一组 Agent 协作审查代码，主动搜寻潜在 Bug。对于每天处理大量 PR 的团队来说，这等于免费多了一个永不疲倦的高级 reviewer。已经可以用了，去试。(33,529 likes | 2,545 RTs) [详情 →](https://x.com/claudeai/status/2031088171262554195)

### Claude Code Desktop 支持定时任务了。
新命令 `/loop` 让你设定最长 3 天的循环任务。典型用法："帮我盯着所有 PR，build 挂了自动修，有评论就开 worktree 处理"；或者"每天早上用 Slack MCP 给我总结被 @  的消息"。这不是 cron job — 是一个有上下文理解的 Agent 在持续工作。(12,790 likes | 838 RTs) [详情 →](https://x.com/bcherny/status/2030193932404150413)

### AlphaGenome 登上 Nature，AI 开始读懂 DNA。
**Google DeepMind** 发布 **AlphaGenome**，能预测基因变异的分子影响、辅助新生物学发现。这不是"又一个生物信息学模型" — 是 AlphaFold 级别的、在 Nature 正刊发表的突破。AI 在生命科学的应用正在加速。(3,444 likes | 737 RTs) [详情 →](https://x.com/GoogleDeepMind/status/2016542480955535475)

**GPT-5.3 Codex 发布。** OpenAI 最新代码模型上线，具体跑分和能力细节待披露。和 Claude Code Review 同期发布，代码 Agent 赛道的竞争愈发激烈。 [详情 →](https://openai.com/index/introducing-gpt-5-3-codex/)

## 🎓 模型小课堂

**代码审查 Agent（Code Review Agent）**：传统代码审查是人逐行看 diff，费时且容易漏。代码审查 Agent 的思路不同 — 它不只是做静态分析，而是像一个有代码理解能力的 reviewer：能追踪变量流向、理解业务逻辑上下文、甚至模拟运行来判断是否有潜在 Bug。多个 Agent 协作审查（如 Claude 的方案）进一步提升覆盖率，每个 Agent 专注不同维度：安全、性能、逻辑正确性。
