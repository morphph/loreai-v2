# Memory is now available on the free plan.  We've also made i，意味着什么

## 🧠 发布动态

### Claude 记忆功能免费开放。
不再是付费专属 — **Claude** 的 Memory 功能现在对所有用户可用，同时支持导入和导出记忆数据。这意味着免费用户也能让 Claude 记住你的偏好、工作背景和常用指令，对话质量会有质的飞跃。已有账号的直接去设置里开启。(38,414 likes | 2,711 RTs) [详情 →](https://x.com/claudeai/status/2028559427167834314)

### Claude Code 上线 Code Review — Agent 帮你审 PR。
一组 Agent 对每个 PR 做深度审查，不是简单的 lint 检查。Anthropic 内部先用了几周，反馈是"能抓到人眼容易漏掉的真实 bug"。背景是今年 Anthropic 工程师人均代码产出涨了 200%，Review 成了瓶颈，所以干脆让 Agent 来分担。团队协作的开发者值得第一时间试试。(5,374 likes | 328 RTs) [详情 →](https://x.com/bcherny/status/2031089411820228645)

### Claude Code 桌面端支持本地定时任务。
设定一个 schedule，Claude Code 会按时自动执行任务 — 只要你的电脑没休眠。想象一下：每天早上自动跑测试、每小时检查日志、定期生成报告。把重复性工作从脑子里卸载掉。(13,547 likes | 1,022 RTs) [详情 →](https://x.com/trq212/status/2030019397335843288)

**Qwen 3.5 系列登场。** 阿里发布 **Qwen3.5-9B** 和 **Qwen3.5-4B**，均为多模态（image-text-to-text）模型，HuggingFace 上线即爆。9B 版本已有 122 万次下载，4B 版本 56 万次。国产开源多模态模型持续卷，小尺寸跑本地的需求被精准拿捏。(680 likes | 1.22M downloads) [详情 →](https://huggingface.co/Qwen/Qwen3.5-9B)

## 🎓 模型小课堂

**Agent 定时任务（Scheduled Tasks）**：传统的定时任务（cron job）需要你写好脚本、配好环境、处理各种边界情况。Agent 定时任务的不同之处在于，你用自然语言描述"我想每天早上检查一下有没有新的 PR 需要审"，Agent 自己理解意图、调用工具、处理异常。这本质上是把"自动化"的门槛从"会写脚本"降到了"会说话"。
