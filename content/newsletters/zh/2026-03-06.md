# GPT-5.4 和 Sonnet 4.6 同日登场，三巨头一周内全部换弹

**2026 年 3 月 6 日**

OpenAI、Anthropic、Google 本周集体刷新模型线 — **GPT-5.4** 带着原生电脑操控和 100 万 token 上下文来了，**Claude Sonnet 4.6** 全面升级编码和 Agent 能力，**Gemini 3.1 Flash-Lite** 则在推理成本上卷到了新低。开发者又赢麻了。

今天聊：三大模型更新、Claude Code 远程控制有多香、以及 Anthropic 抓到三家中国 AI 公司大规模蒸馏模型。

## 🧠 发布动态

### GPT-5.4 来了，这次带了电脑操控。
OpenAI 一口气推出 **GPT-5.4 Thinking** 和 **GPT-5.4 Pro**，API 和 Codex 同步上线。100 万 token 上下文、原生 Computer Use、Agent 编码能力全面增强 — 这是对 Claude 电脑操控和 Gemini 长上下文的正面回应。ChatGPT 用户现在就能体验，API 开发者直接切换模型 ID。(19,781 likes | 2,796 RTs) [详情 →](https://x.com/OpenAI/status/2029620619743219811)

### Claude Sonnet 4.6 全面刷新。
Anthropic 更新 Claude 4 家族中坚力量 — **Sonnet 4.6** 在编码、Agent 和专业任务上达到 frontier 水平，性价比拉满。如果你的产品在用 Sonnet，现在就该切到 `claude-sonnet-4-6`。 [详情 →](https://www.anthropic.com/news/claude-sonnet-4-6)

### Gemini 3.1 Flash-Lite：推理成本卷王。
Google 最快、最便宜的 **Gemini 3** 系列模型上线，专攻大规模生产负载。在推理成本逐底竞赛中，Flash-Lite 瞄准的是你每天跑几百万次调用的那些场景。拿来跟你现在用的廉价模型跑个对比。(7,751 likes | 727 RTs) [详情 →](https://deepmind.google/blog/gemini-3-1-flash-lite-built-for-intelligence-at-scale/)

**Phi-4-reasoning-vision-15B 开源。** 微软给 **Phi-4** 家族加上了视觉推理能力，15B 参数 — 小模型 + 强推理 + 视觉理解，这是端侧部署的甜蜜点。(72 likes | 6.8K downloads) [详情 →](https://huggingface.co/microsoft/Phi-4-reasoning-vision-15B)

**NVIDIA 发布 Qwen3.5-397B FP4 量化版。** 397B 参数的 MoE 大模型，用 NVFP4 量化后只有 17B 活跃参数 — 原来需要一堆卡才能跑的巨兽，现在门槛大幅降低。(61 likes | 77.5K downloads) [详情 →](https://huggingface.co/nvidia/Qwen3.5-397B-A17B-NVFP4)

## 🔧 开发者工具

### Claude Code 支持远程控制了。
在终端启动一个任务，掏出手机继续盯 — **Claude Code** 现在支持会话持久化和远程接管。跑一个大规模重构然后出门遛弯，手机上看进度、随时干预。长时间 Agent 编码任务的体验直接上了一个台阶。(44,212 likes | 4,660 RTs) [详情 →](https://x.com/claudeai/status/2026418433911603668)

**Claude Cowork 支持定时任务。** 设好时间，Claude 自动执行重复性工作 — 每天早上生成简报、每周更新报表、周五做 PPT。从"你问它答"变成"它主动干活"。(22,272 likes | 1,696 RTs) [详情 →](https://x.com/claudeai/status/2026720870631354429)

**OpenAI Responses API 上线 WebSocket。** 持久连接为重度工具调用的 Agent 工作流大幅降低延迟。如果你的 Agent 每轮对话要调十几个工具，从轮询切到 WebSocket 会有质的提升。(5,028 likes | 346 RTs) [详情 →](https://x.com/OpenAIDevs/status/2026025368650690932)

**openai-python v2.26.0**：`ComputerTool` 正式 GA，替换之前的 preview API。在做电脑操控 Agent 的赶紧迁移，旧接口要下线了。 [详情 →](https://github.com/openai/openai-python/releases/tag/v2.26.0)

**Transformers v5.3.0**：一次加 7 个新架构 — **EuroBERT**（多语言编码器）、**VibeVoice ASR**（语音识别）、**TimesFM2.5**（时序预测）等。`pip install --upgrade transformers` 走起。 [详情 →](https://github.com/huggingface/transformers/releases/tag/v5.3.0)

## 🎓 模型小课堂

**模型蒸馏（Model Distillation）**：简单说就是"学霸教学渣" — 用一个大模型（老师）的输出来训练一个小模型（学生），让小模型学到大模型的能力但体积小得多、推理快得多。正经用法是压缩部署成本，但今天 Anthropic 披露的情况是：有人拿你的商业模型当免费老师，大规模生成训练数据来训自家模型。这就从技术手段变成了知识产权问题。
