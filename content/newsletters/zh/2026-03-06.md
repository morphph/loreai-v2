# OpenAI、Anthropic、Google 同一天发模型，三国杀正式开打

**2026 年 3 月 6 日**

今天是 2026 年 AI 军备竞赛最疯狂的一天 — **GPT-5.4** 带着原生电脑操控和 100 万 token 上下文上线，**Claude Sonnet 4.6** 同步登场，**Gemini 3.1 Flash-Lite** 卷价格。三巨头同日交火，开发者又赢麻了。

今天聊：三大模型同日对决、Anthropic 实锤中国三家公司蒸馏攻击、以及 Claude Code 可以用手机遥控了。

## 🧠 发布动态

### GPT-5.4 来了，原生电脑操控正式 GA。
OpenAI 最新旗舰模型 — 100 万 token 上下文、原生 Computer Use（不再是 preview）、知识工作和代码能力全面提升。Sam Altman 特别强调了"中途转向"能力：你可以在模型推理过程中改变指令方向。API 和 Codex 已可用，ChatGPT 也在滚动上线 Thinking 和 Pro 版本。(19,781 likes | 2,796 RTs) [详情 →](https://x.com/OpenAI/status/2029620619743219811)

### Claude Sonnet 4.6 上线。
Anthropic 4.6 家族补齐速度档 — Sonnet 4.6 在编码、Agent 和专业场景达到 frontier 水平，同时保持 Sonnet 系列一贯的高性价比。如果你之前用 Sonnet 4.5 跑量，现在该切了。 [详情 →](https://www.anthropic.com/news/claude-sonnet-4-6)

### Gemini 3.1 Flash-Lite：Google 卷到底。
**Google** 推出 Gemini 3 系列最便宜的模型，瞄准高吞吐生产场景。三家同一天发模型不是巧合 — token 单价的军备竞赛已经和模型能力的军备竞赛一样激烈。做推理密集型应用的，现在该把三家都跑个 benchmark 了。(7,751 likes | 727 RTs) [详情 →](https://deepmind.google/blog/gemini-3-1-flash-lite-built-for-intelligence-at-scale/)

**Phi-4-reasoning-vision-15B**：**微软** 给 Phi-4 推理系列加上了视觉能力，15B 参数就能对图像做推理。边缘部署和成本敏感场景的福音 — 小模型多模态推理这条路越走越宽。(72 likes | 6.8K downloads) [详情 →](https://huggingface.co/microsoft/Phi-4-reasoning-vision-15B)

## 🎓 模型小课堂

**模型蒸馏（Model Distillation）**：简单说就是用大模型"教"小模型。你拿一个很强的大模型生成大量高质量输出，然后让小模型学习模仿这些输出，最终小模型能达到接近大模型的效果但成本低得多。问题在于：如果你用别人的 API 偷偷做这件事，本质上是在"偷"别人几亿美元的训练成果 — 这就是 Anthropic 今天点名几家公司的核心争议。
