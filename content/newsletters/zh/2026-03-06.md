# GPT-5.4 来了，Claude Code 能说话了，Anthropic 抓到三家公司偷模型

**2026 年 3 月 6 日**

今天最炸的消息不是新模型 — 而是 Anthropic 公开点名 **DeepSeek**、**Moonshot AI**、**MiniMax** 大规模蒸馏 Claude，2.4 万假账号、1600 万次对话，工业级偷模型。与此同时，新模型照样卷：**GPT-5.4** 上线，**GLM-5** 开源，**Qwen3-Coder-Next** 下载破百万。

## 🔬 研究前沿

### 基础设施噪声正在污染你的跑分 — 这个发现很重要。
Anthropic 工程团队发现，服务器配置差异能让 Agent 编码 eval 波动好几个百分点 — 有时比顶级模型之间的差距还大。换句话说：如果你根据排行榜上 2% 的差距选模型，你选的可能是服务器配置而不是模型能力。在自己的场景上跑 eval，别迷信排行榜。 [详情 →](https://www.anthropic.com/engineering/infrastructure-noise)

**蒸馏攻击检测技术细节公开。** Anthropic 发布了检测方法和防御措施的技术报告 — 如果你在运营模型 API，这是必读材料。核心思路：行为模式分析 + 异常调用检测。 [详情 →](https://www.anthropic.com/news/detecting-and-preventing-distillation-attacks)

## 🎓 模型小课堂

**模型蒸馏（Model Distillation）**：简单说就是用大模型教小模型 — 把大模型的输入输出当"教材"，训练一个更小更快的模型来模仿大模型的行为。合法的蒸馏是 AI 研究的标准技术，OpenAI 和 Google 自己也在用。但 Anthropic 今天披露的问题是：有人大规模伪造账号，未经授权地用 Claude 的 API 输出来训练自己的模型 — 这不是蒸馏，这是盗版。
