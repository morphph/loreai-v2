# 三巨头同一周发模型，AI 军备竞赛卷出新高度

**2026 年 3 月 6 日**

OpenAI、Anthropic、Google 本周集体交作业 — **GPT-5.4**、**Claude Sonnet 4.6**、**Gemini 3.1 Flash-Lite** 扎堆发布，frontier 模型的价格战和性能战同时打响。与此同时，Anthropic 公开指控三家中国 AI 公司大规模蒸馏 Claude，行业暗战浮出水面。

今天聊：三巨头模型对决、Claude Code 跨设备遥控、以及 Karpathy 又把训练时间砍了一刀。

## 🧠 发布动态

### GPT-5.4 来了，原生 Computer Use + 百万 token 上下文。
OpenAI 几个月来最大的模型更新 — **GPT-5.4 Thinking** 和 **GPT-5.4 Pro** 同时上线 ChatGPT。原生计算机操作、Codex/API 支持 100 万 token 上下文、推理能力强化，直接对标 Claude Sonnet 4.6 和 Gemini 3.1。如果你在做 Agent 工作流，现在就去 API 里跑一圈对比。(19,781 likes | 2,796 RTs) [详情 →](https://x.com/OpenAI/status/2029620619743219811)

### Claude Sonnet 4.6 — Anthropic 说 Claude 是"思考的空间"。
Sonnet 4.6 在编码、Agent 和专业任务上达到 frontier 水准。但更值得关注的是产品定位的转变：Anthropic 不再把 Claude 当聊天工具卖，而是定义为"思考伙伴"。这不只是营销话术 — 产品交互逻辑会跟着变。拿你最难的推理任务去试试。[详情 →](https://www.anthropic.com/news/claude-is-a-space-to-think)

### Gemini 3.1 Flash-Lite：Google 的性价比杀手。
**Google** 放出 Gemini 3 系列中成本最低的模型，瞄准大规模生产负载。三家 frontier 实验室同一周发模型 — 成本-性能曲线正在被快速压低，用便宜模型跑量的场景该重新评估了。(7,751 likes | 727 RTs) [详情 →](https://deepmind.google/blog/gemini-3-1-flash-lite-built-for-intelligence-at-scale/)

**Transformers v5.3.0 一口气加了七个新架构。** EuroBERT（多语言双向编码）、VibeVoice（语音识别）、TimesFM2.5（时序预测）… HuggingFace 的野心是让 transformers 库覆盖一切。更新依赖看看有没有能直接替换你自定义模型的。[详情 →](https://github.com/huggingface/transformers/releases/tag/v5.3.0)

## 🎓 模型小课堂

**MoE（Mixture of Experts，混合专家模型）**：传统模型每次推理都要激活所有参数，MoE 的思路是把模型拆成多个"专家"模块，每次只激活其中几个。比如 Qwen3.5-35B-A3B 有 350 亿总参数，但每次推理只用 30 亿 — 成本降了一个数量级，能力却没怎么掉。代价是总参数量（也就是模型文件大小）更大，但如果你更在意推理速度和显存占用，MoE 是目前最实用的"既要又要"方案。
