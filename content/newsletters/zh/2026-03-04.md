# OpenAI 拿下 1100 亿美元，Claude 登顶 App Store — AI 的钱和用户都在加速集中

**2026 年 3 月 4 日**

OpenAI 刚完成史上最大创业融资 — 1100 亿美元，估值 8400 亿。与此同时，**Claude** 悄悄爬上了 App Store 第一名。一边是资本疯狂下注，一边是用户用脚投票，AI 战场格局正在快速重塑。

今天聊：天价融资背后的信号、Claude Code 的两个新杀手技能、以及 Gemini 3.1 Pro 和 Qwen 3.5 的新动态。

---

## 🧠 模型动态

**Gemini 3.1 Pro 来了。** Google DeepMind 发布最新旗舰推理模型，定位"简单回答不够用"的复杂任务 — 多步推理和长文档分析有明显提升。Claude Opus、GPT-5 的三方对决又多了新变量。 [详情 →](https://deepmind.google/blog/gemini-3-1-pro-a-smarter-model-for-your-most-complex-tasks/)

**Qwen3.5-35B-A3B 上线 HuggingFace。** 通义千问最新 MoE 模型，35B 总参数只激活 3B — 多模态图文能力，下载量已破 68 万。对资源有限的开发者来说，这个参数效率很有吸引力。 [详情 →](https://huggingface.co/Qwen/Qwen3.5-35B-A3B)

---

## 🔧 开发者工具

**Claude Code 两个新技能：/simplify 和 /batch。** /simplify 自动精简 PR，把代码从"能跑"优化到"该合"；/batch 并行处理批量代码迁移。Anthropic 工程师自己天天在用，下个版本上线。每天要 review 大量 PR 的团队会很有感。 [详情 →](https://x.com/bcherny/status/2027534984534544489)

**AskUserQuestion 支持 Markdown 了。** Claude Code 交互中可以展示代码片段、图表、表格，Agent 开发体验又顺滑了一点。 [详情 →](https://x.com/bcherny/status/2027627119371817190)

**Bun v1.3.10 发布，修复 155 个问题。** 新增原生 JS/TS REPL、Windows ARM64 支持、ECMAScript 装饰器。Bun 的完成度越来越高。 [详情 →](https://x.com/bcherny/status/2027627970731643362)

## 📝 技术前沿

**你的 Agent 跑分可能被基础设施噪声带偏了。** Anthropic 工程团队量化发现，服务器配置差异能让 Agent 编码评测结果波动好几个百分点 — 有时比排行榜上顶级模型的差距还大。下次看跑分排行，心里打个折扣。 [详情 →](https://www.anthropic.com/engineering/infrastructure-noise)

**怎么出 AI 做不了的技术面试题？** 当 Claude 能轻松通过你的 take-home 时，面试官该怎么办？Anthropic 分享了三轮迭代的方法论，招聘流程该更新了。 [详情 →](https://www.anthropic.com/engineering/AI-resistant-technical-evaluations)

**MCP 还是 CLI？** 一篇引发热议的文章：MCP 在很多场景下是过度工程，传统 CLI 工具反而更简单有效。做 Agent 架构选型的值得读一下。 [详情 →](https://ejholmes.github.io/2026/02/28/mcp-is-dead-long-live-the-cli.html)
