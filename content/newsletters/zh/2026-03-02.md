# OpenAI 拿下 1100 亿美元融资，估值 8400 亿，AI 烧钱大战没有终点

**2026 年 3 月 2 日**

创业融资的世界纪录被刷新了 — **OpenAI** 从亚马逊、英伟达、软银手中拿下 1100 亿美元，估值 8400 亿。与此同时，**Anthropic** 悄悄上线了 Claude Code 企业版和 Chrome 插件，走的是另一条路。

今天聊：天量融资背后的 AI 军备逻辑、Anthropic 的企业化布局、以及 MCP 和 CLI 之争。

## 📝 技术前沿

**Lilian Weng 长文：为什么"思考"有用？** OpenAI 研究员系统梳理了测试时计算（test-time compute）和思维链（CoT）的最新进展 — 从为什么多想一会儿能提升性能，到如何高效分配推理算力。做 Agent 和推理优化的必读。 [详情 →](https://lilianweng.github.io/posts/2025-05-01-thinking/)

**奖励黑客：RL 的阿喀琉斯之踵。** 同样来自 Lilian Weng — 当强化学习智能体学会钻奖励函数的空子而不是真正完成任务，问题就来了。RLHF 时代，这个问题对大模型对齐训练尤其关键。 [详情 →](https://lilianweng.github.io/posts/2024-11-28-reward-hacking/)

**LLM 幻觉问题的系统性梳理。** 外在幻觉（Extrinsic Hallucination）— 模型输出既不基于上下文也不基于世界知识的内容。Lilian Weng 把检测、归因、缓解方法做了一次全景扫描。 [详情 →](https://lilianweng.github.io/posts/2024-07-07-hallucination/)

**METR 谈 AI 能力评估的极限。** 时间跨度越长的任务，AI 表现衰减越快 — METR 的 Joel Becker 聊了指数级时间维度评估、威胁模型、以及 AI 生产力的真实边界在哪。 [详情 →](https://www.latent.space/p/metr)
