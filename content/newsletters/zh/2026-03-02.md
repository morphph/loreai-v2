# OpenAI 拿下 1100 亿美元史上最大融资，Google 一天三连发

**2026 年 3 月 2 日**

**OpenAI** 以 8400 亿美元估值融了 1100 亿美元 — 亚马逊、英伟达、软银联手押注，创业公司融资纪录直接改写。**Google** 也不闲着，一口气发了 Gemini 3.1 Pro、Nano Banana 2 图像生成、还让 Gemini 能作曲了。

今天聊：天价融资、Google 三连发、以及 Claude 生态这周的密集更新。

---

## 🧠 模型动态

**Gemini 3.1 Pro 来了。** Google 最新旗舰推理模型，主打复杂任务 — 不只给答案，给完整的分析链条。和 Opus 4.6、GPT-5.3 正面对线。 [详情 →](https://deepmind.google/blog/gemini-3-1-pro-a-smarter-model-for-your-most-complex-tasks/)

**Qwen3.5-35B-A3B 上线 HuggingFace。** 阿里最新多模态 MoE 模型，350 亿参数但激活仅 30 亿，HuggingFace 上已超 68 万次下载。又轻又能打，本地部署党的福音。 [详情 →](https://huggingface.co/Qwen/Qwen3.5-35B-A3B)

**Nano Banana 2 发布。** Google 新图像生成模型，Pro 级能力 + Flash 级速度，支持主体一致性和高级世界知识。DALL-E 和 Midjourney 又多了个对手。 [详情 →](https://deepmind.google/blog/nano-banana-2-combining-pro-capabilities-with-lightning-fast-speed/)

## 🔧 开发者工具

**Claude Code 新增 /simplify 和 /batch。** /simplify 自动精简 PR 到可合并状态，/batch 并行处理代码迁移任务。两个高频痛点，一次解决。 [详情 →](https://x.com/bcherny/status/2027534984534544489)

**Claude Code Remote 对 Pro 用户开放。** 不用本地跑了，远程执行 — 没有高配开发机的团队终于不用眼馋了。 [详情 →](https://x.com/bcherny/status/2027462787358949679)

**AskUserQuestion 工具升级。** 现在能展示 Markdown 片段、图表和代码块。Agent 和人之间的沟通终于不是纯文本了。 [详情 →](https://x.com/bcherny/status/2027627119371817190)

## 📝 技术前沿

**用一队并行 Claude 造了个 C 编译器。** Anthropic 让 Opus 4.6 的 Agent Teams 自主构建编译器，然后基本放手不管。结论：多智能体协作已经能搞定相当复杂的工程任务了。 [详情 →](https://www.anthropic.com/engineering/building-c-compiler)

**基础设施噪声正在污染你的跑分。** 服务器配置差异能让 Agent 编码评测分数波动好几个百分点 — 有时比顶级模型之间的差距还大。排行榜的可信度要打折了。 [详情 →](https://www.anthropic.com/engineering/infrastructure-noise)

**Lilian Weng 万字长文：Why We Think。** 系统梳理 test-time compute 和 CoT（思维链）的最新进展 — 为什么"让模型多想一会儿"能这么有效？想深入理解推理模型的必读。 [详情 →](https://lilianweng.github.io/posts/2025-05-01-thinking/)
