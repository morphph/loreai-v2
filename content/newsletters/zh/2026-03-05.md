# OpenAI 1100 亿美元融资落地，Claude 霸榜 App Store — AI 的钱和人都在加速流动

**2026 年 3 月 5 日**

**OpenAI** 完成史上最大创业公司融资 — 1100 亿美元，估值 8400 亿。与此同时，**Claude** 悄悄爬上了 App Store 第一名。一边是资本的狂欢，一边是用户的投票，AI 竞争的两个维度从未如此清晰。

今天聊：天价融资背后的信号、Google 一口气发三个模型、以及 Claude Code 占了 GitHub 4% 的提交量。

## 🧠 模型动态

**Gemini 3.1 Pro 来了。** Google 发布最新旗舰推理模型，主打"简单答案不够用"的复杂任务场景。和 Opus 4.6、GPT-5.3 正面交锋的意思很明显。 [详情 →](https://deepmind.google/blog/gemini-3-1-pro-a-smarter-model-for-your-most-complex-tasks/)

**Qwen3.5-35B-A3B 上线。** 阿里的多模态模型（图文理解），35B 参数但激活仅 3B — MoE 架构让推理成本大幅降低。HuggingFace 上线不久已有 68 万次下载。 [详情 →](https://huggingface.co/Qwen/Qwen3.5-35B-A3B)

**Nano Banana 2 发布。** Google 图像生成新模型，把 Pro 级能力压到 Flash 速度。主体一致性、世界知识、生产级规格一个不少。 [详情 →](https://deepmind.google/blog/nano-banana-2-combining-pro-capabilities-with-lightning-fast-speed/)

## 🔧 开发者工具

**Claude Code 新增 /simplify 和 /batch 两个 Skill**：前者自动化代码审查和 PR 质量把控，后者处理批量代码迁移。开发者日常最费时间的两件事，现在一键搞定。 [详情 →](https://x.com/bcherny/status/2027534984534544489)

**Claude Code AskUserQuestion 支持 Markdown 预览**：可以直接展示代码片段、ASCII 图表和设计方案供用户选择。Agent 交互体验又精细了一层。 [详情 →](https://x.com/trq212/status/2027627119371817190)

**Bun v1.3.10 大更新**：一次修了 155 个 issue（642 个点赞），新增原生 JS/TS REPL、Windows ARM64 支持、ECMAScript 装饰器。 [详情 →](https://x.com/bcherny/status/2027627970731643362)

## 📝 技术前沿

**基础设施噪声正在污染你的 Agent 跑分。** Anthropic 工程团队量化发现，服务器配置差异能让编码评测得分波动好几个百分点 — 有时比排行榜上顶级模型的差距还大。以后看跑分，先问一句"在什么环境跑的"。 [详情 →](https://www.anthropic.com/engineering/infrastructure-noise)

**AI 能通过你的面试题？Anthropic 分享"防 AI 面试"设计经验。** 当 Claude 能轻松搞定 take-home exam，招聘团队怎么办？三轮迭代的实战经验值得每个技术主管看看。 [详情 →](https://www.anthropic.com/engineering/AI-resistant-technical-evaluations)

**Lilian Weng 连发三篇深度综述**：强化学习中的奖励 hack、大语言模型（LLM）外在幻觉、视频生成扩散模型。每篇都是教科书级别的梳理，适合存档慢读。 [详情 →](https://lilianweng.github.io/posts/2024-11-28-reward-hacking/)

**Knuth 写了篇关于 Claude 的论文。** 计算机科学泰斗 Donald Knuth 发表 "Claude's Cycles"，斯坦福出品。当 Knuth 开始研究 LLM，说明这个领域的学术严肃性又上了一个台阶。 [详情 →](https://www-cs-faculty.stanford.edu/~knuth/papers/claude-cycles.pdf)
