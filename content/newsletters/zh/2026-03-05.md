# Claude 登顶 App Store，Google 一口气甩出四个 Gemini 更新

**2026 年 3 月 5 日**

**Claude** 冲上美区 App Store 第一，**OpenAI** 拿下史上最大单轮融资 1100 亿美元，**Google** 同一天连发 Gemini 3.1 Pro、Gemini 3 Deep Think、Nano Banana 2 和音乐生成 — 三家巨头同时发力，开发者的选择焦虑又升级了。

今天聊：Claude 生态的全面爆发、Google 的模型矩阵轰炸、以及 OpenAI 那个让人瞠目的估值。

---

## 🧠 模型动态

**Gemini 3.1 Pro 发布。** Google 最新旗舰推理模型，主打复杂任务场景 — 当一个简单答案不够用的时候。和 Claude Opus 4.6 正面对标的意思很明显。 [详情 →](https://deepmind.google/blog/gemini-3-1-pro-a-smarter-model-for-your-most-complex-tasks/)

**Nano Banana 2 来了。** Google 图像生成新模型，Pro 级能力 + Flash 级速度，支持世界知识、主体一致性，直接可用于生产环境。名字很怪，东西挺硬。 [详情 →](https://deepmind.google/blog/nano-banana-2-combining-pro-capabilities-with-lightning-fast-speed/)

**Gemini 3 Deep Think 更新。** 专攻科学研究和工程推理的深度思考模式，Google 在垂直推理上继续加码。 [详情 →](https://deepmind.google/blog/gemini-3-deep-think-advancing-science-research-and-engineering/)

**Qwen3.5-35B-A3B 上线 HuggingFace。** 阿里通义千问新版本，35B 参数但仅激活 3B 的 MoE 架构，多模态能力，下载量已超 68 万。轻量高效这条路，国产模型走得越来越成熟。 [详情 →](https://huggingface.co/Qwen/Qwen3.5-35B-A3B)

## 🔧 开发者工具

**Claude Code 新技能：/simplify 和 /batch。** 两个即将上线的 Skills — /simplify 自动审查代码质量并修复，/batch 并行处理批量代码迁移。从 PR 到上线的流程又短了一截。 [详情 →](https://x.com/bcherny/status/2027534984534544489)

**Claude Code AskUserQuestion 支持 Markdown 渲染。** 现在可以在交互中展示图表、代码片段 — Agent 和人的沟通终于不只是纯文本了。 [详情 →](https://x.com/bcherny/status/2027627119371817190)

**Bun v1.3.10 发布。** 一次性修复 155 个问题，新增原生 JS/TS REPL、Windows ARM64 支持、ECMAScript 装饰器。用 Bun 的同学可以更新了。 [详情 →](https://x.com/bcherny/status/2027627970731643362)

## 📝 技术前沿

**基础设施噪声正在污染你的 Agent 跑分。** Anthropic 工程团队量化了一个被忽视的问题：服务器配置差异导致的评测波动，有时比模型之间的真实差距还大。下次看排行榜，记得打个折扣。 [详情 →](https://www.anthropic.com/engineering/infrastructure-noise)

**Donald Knuth 写了篇关于 Claude 的论文。** 斯坦福计算机科学泰斗 Knuth 发布 "Claude's Cycles"，Hacker News 上 636 赞。大师怎么看 AI，值得一读。 [详情 →](https://www-cs-faculty.stanford.edu/~knuth/papers/claude-cycles.pdf)

**Gemini 能作曲了。** 基于 Lyria 3 模型，文字或图片输入即可生成 30 秒音乐片段。多模态的边界又往外推了一步。 [详情 →](https://deepmind.google/blog/a-new-way-to-express-yourself-gemini-can-now-create-music/)

**Lilian Weng 新文：视频生成的扩散模型。** 从图像到视频，时间一致性和世界知识编码是核心挑战。系统性综述，适合周末深读。 [详情 →](https://lilianweng.github.io/posts/2024-04-12-diffusion-video/)
