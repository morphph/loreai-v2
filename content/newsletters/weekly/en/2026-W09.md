# 5 Things That Mattered in AI This Week

**Week of February 23 to February 28, 2026**

The AI price war entered a new phase while open-source reasoning models reached unprecedented heights. Here are the five stories that defined the week.

---

## 1. OpenAI Launches GPT-5 Turbo at Half the Price

OpenAI's distilled variant of GPT-5 shipped this week at $2.50/M input and $10/M output -- a 50% price cut from the full model. GPT-5 Turbo matches GPT-5 on 94% of benchmarks while running significantly faster, making it the new default for most API users.

**Why it matters:** This is the most aggressive pricing move since the GPT-4 Turbo launch. The margin compression across the industry is relentless -- every major lab is now racing to offer frontier-quality reasoning at commodity prices. For developers, the practical implication is clear: the cost barrier for building sophisticated AI applications is dropping faster than most product roadmaps can keep up with.

**What's next:** Expect Google and Anthropic to respond with competitive pricing within weeks. The real question is whether distillation techniques will continue yielding 2x efficiency gains per generation, or if we're approaching diminishing returns.

[Read more ->](https://openai.com/index/gpt-5-turbo/)

---

## 2. DeepSeek R2 Drops as Open-Weight Reasoning Champion

DeepSeek's follow-up to R1 scored 89.4% on AIME 2025 and 71.2% on Codeforces, putting it ahead of o3-mini on most reasoning benchmarks. The model uses a massive 685B total parameter MoE architecture with 37B active parameters, released under MIT license.

**Why it matters:** For the first time, an open-weight model has convincingly beaten a proprietary reasoning model on multiple benchmarks. Running DeepSeek R2 costs about $8/hour on 8xH100, making it accessible for well-funded startups and research labs. The Chinese AI ecosystem continues to produce models that challenge the assumption that only US labs can build frontier systems.

**What's next:** The community is already fine-tuning R2 for specialized domains. Expect a wave of reasoning-focused applications built on R2 in the coming months, particularly in code generation and mathematical problem-solving.

[Read more ->](https://github.com/deepseek-ai/DeepSeek-R2)

## 3. Claude Code Launches Remote Control for Mobile Devices

Anthropic introduced Remote Control for Claude Code, allowing developers to manage their AI coding sessions from a phone or tablet while the agent runs on a remote machine. The feature uses secure WebSocket connections and requires no additional setup beyond the existing Claude Code installation.

**Why it matters:** This is the first major step toward truly asynchronous AI-assisted development. Developers can kick off complex refactoring or feature implementation tasks and monitor progress from anywhere. It fundamentally changes the workflow from "pair programming with AI" to "delegating to AI and reviewing results."

**What's next:** The obvious extension is full CI/CD integration -- imagine Claude Code running as a first-pass reviewer on every pull request, with developers approving changes from their phone during their morning commute.

[Read more ->](https://code.claude.com/docs/en/remote-control)

---

## 4. Apple Intelligence Opens to Third-Party Plugins

Apple announced that third-party developers can now build plugins for Apple Intelligence, extending Siri's capabilities with custom AI models and data sources. The plugin SDK supports on-device and cloud-based processing, with Apple's privacy review process for all submissions.

**Why it matters:** Apple's 2 billion active devices represent the largest potential distribution channel for AI applications. The plugin system creates a new app store category where AI capabilities, not user interfaces, are the primary product. For developers building AI tools, Apple's distribution reach could dwarf anything available through web-based platforms.

**What's next:** The first wave of plugins will likely focus on productivity (email summarization, calendar management) and creative tools (image generation, writing assistance). The real disruption comes when domain-specific AI models -- medical, legal, financial -- gain access to Apple's user base.

[Read more ->](https://developer.apple.com/apple-intelligence/plugins)

---

## 5. Meta Open-Sources Llama 4 Scout

Meta released Llama 4 Scout, a 17B active / 109B total parameter MoE model that achieves GPT-4-class performance on standard benchmarks. The model includes a massive 10M token context window and supports multimodal inputs (text, image, video).

**Why it matters:** Llama 4 Scout's 10M context window is the largest available in any open model, making it immediately useful for document processing, codebase analysis, and long-form content generation. The MoE architecture keeps inference costs manageable despite the large total parameter count. Meta's continued investment in open models maintains competitive pressure on proprietary labs.

**What's next:** The extended context window opens up new application categories that were previously impossible with open models. Expect rapid adoption in enterprise settings where data privacy requirements make proprietary APIs impractical.

[Read more ->](https://ai.meta.com/blog/llama-4-scout/)
