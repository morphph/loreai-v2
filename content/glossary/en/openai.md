# OpenAI — AI Glossary

**OpenAI** is an AI research and deployment company that builds large language models, image generators, and developer APIs. Founded in 2015 as a nonprofit research lab, it transitioned to a capped-profit structure in 2019 and is best known for creating the GPT series of models, ChatGPT, and DALL-E.

## Why OpenAI Matters

OpenAI catalyzed the current wave of generative AI adoption. ChatGPT's launch in November 2022 brought large language models into mainstream use, reaching 100 million users faster than any consumer application before it. The company's API platform powers thousands of products — from coding assistants to customer support bots to content generation tools.

For developers, OpenAI's API set the de facto standard for LLM integration: chat completions, function calling, embeddings, and fine-tuning endpoints that most competing providers now mirror. Understanding OpenAI's ecosystem is essential context for evaluating alternatives like [Claude](/glossary/claude) and other foundation models.

## How OpenAI Works

OpenAI develops foundation models using transformer architectures trained on large-scale datasets. Its product stack spans several layers:

- **Models**: The GPT family (GPT-4, GPT-4o, o1, o3) for text generation and reasoning, plus DALL-E for image generation and Whisper for speech recognition
- **ChatGPT**: A consumer-facing chat interface available as a web app, mobile app, and desktop client, with free and paid tiers
- **API Platform**: REST APIs for model inference, fine-tuning, embeddings, and assistants — the primary integration point for developers
- **Ecosystem**: Plugins, GPTs (custom chatbot configurations), and an enterprise tier with enhanced privacy and admin controls
