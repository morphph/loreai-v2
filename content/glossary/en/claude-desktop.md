# Claude Desktop — AI Glossary

**Claude Desktop** is Anthropic's native desktop application for macOS and Windows that provides direct access to Claude outside the browser. It supports text conversations, image and document analysis, file uploads, and — with its **Computer Use** and **Cowork** capabilities — agentic workflows where Claude can observe and interact with your screen. Unlike the web interface at claude.ai, the desktop app integrates with your local environment and supports extended background tasks.

## Why Claude Desktop Matters

Claude Desktop bridges the gap between a chat-based AI assistant and an on-device productivity tool. By running natively, it can process local files without uploading them through a browser, handle drag-and-drop workflows, and maintain persistent conversations across sessions.

The addition of **agentic features** — where Claude can view your screen, click UI elements, and operate applications on your behalf — positions Claude Desktop as more than a chatbot. It becomes a general-purpose digital assistant capable of completing multi-step tasks across desktop applications. For coverage of Claude's latest desktop agent capabilities, see our [analysis of Claude's Office integration](/blog/claude-excel-powerpoint-sync). Developers building [agentic coding](/glossary/agentic-coding) workflows can pair Claude Desktop with terminal-based tools for a hybrid approach.

## How Claude Desktop Works

Claude Desktop connects to Anthropic's API while adding a native application layer for local system integration. Key mechanisms:

- **File handling**: Drag files directly into conversations — PDFs, images, spreadsheets, code files — for analysis without manual copy-paste
- **MCP integration**: Connects to local and remote **Model Context Protocol** servers, giving Claude access to databases, APIs, and development tools from the desktop
- **Computer Use**: An opt-in mode where Claude receives screenshots of your display and can execute mouse clicks and keyboard input to operate applications autonomously
- **Persistent projects**: Organize conversations by topic with attached files and custom instructions that carry across sessions

## Related Terms

- **[ChatGPT](/glossary/chatgpt)**: OpenAI's competing conversational AI, also available as a desktop app with broadly similar chat capabilities
- **[Cursor](/glossary/cursor)**: An AI-enhanced code editor — specialized for development, whereas Claude Desktop is a general-purpose assistant
- **[Agentic Coding](/glossary/agentic-coding)**: The paradigm of AI agents autonomously writing and executing code, which Claude Desktop supports through Computer Use and MCP
