# Anthropic — AI 术语表

**Anthropic** 是一家总部位于旧金山的人工智能安全公司，由前 OpenAI 研究副总裁 Dario Amodei 和 Daniela Amodei 于 2021 年联合创立。公司开发了 [Claude](/glossary/claude) 系列大语言模型，是当前 AI 行业最具影响力的实验室之一。

## 为什么 Anthropic 重要

Anthropic 的核心理念是"安全优先"——在推进模型能力的同时，系统性地研究如何让 AI 更可控、更可解释。这种思路直接影响了 Claude 模型的设计：Constitutional AI（宪法式 AI）训练方法让模型在遵循指令的同时保持安全边界。

对开发者而言，Anthropic 提供了完整的产品生态：Claude API 支持文本生成、工具调用和多模态输入，[Claude Code](/glossary/claude-code) 则是面向终端的智能编码代理。Google 是 Anthropic 的主要投资者之一，Amazon 也通过 [AWS Bedrock](/glossary/amazon) 提供 Claude 模型的托管服务。更多关于 Claude 最新能力的分析，参见我们的[报道](/blog/anthropic-claude-memory-upgrades-importing)。

## Anthropic 的核心技术

Anthropic 的技术栈围绕几个关键方向展开：

- **Constitutional AI**：通过一组明确的原则（"宪法"）引导模型行为，减少对人工标注的依赖
- **可解释性研究**：团队在机械可解释性（Mechanistic Interpretability）领域处于前沿，致力于理解模型内部的决策机制
- **长上下文处理**：Claude 模型支持最高 200K token 的上下文窗口，适用于大规模文档分析和代码库理解
- **Agent 能力**：从 [Claude Code](/glossary/claude-code) 到[桌面端智能体](/blog/anthropic-cowork-claude-desktop-agent)，Anthropic 正在将模型从对话工具推向自主执行任务的代理

## 相关术语

- **[Claude](/glossary/claude)**：Anthropic 开发的大语言模型系列，包括 Opus、Sonnet、Haiku 等不同规格
- **[Claude Code](/glossary/claude-code)**：基于 Claude 的终端编码代理，支持项目级代码理解和自主开发
- **[Google](/glossary/google)**：Anthropic 的主要投资方之一，同时也是 Gemini 模型的开发者
