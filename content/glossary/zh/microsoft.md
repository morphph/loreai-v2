# Microsoft — AI 术语表

**Microsoft**（微软）是全球市值最高的科技公司之一，也是当前 AI 竞赛中最具影响力的玩家。通过对 OpenAI 超过 130 亿美元的投资，Microsoft 将 GPT 系列模型深度整合进 Azure 云平台、Microsoft 365 办公套件和 GitHub Copilot 等开发者工具，构建了覆盖云端到终端的 AI 生态。

## 为什么 Microsoft 重要

Microsoft 的 AI 战略改变了整个行业的竞争格局。**Azure OpenAI Service** 让企业客户能在合规环境中直接调用 GPT-4、DALL·E 等模型，降低了大模型落地的门槛。**GitHub Copilot** 是最早大规模商用的 AI 编程助手，推动了 AI 辅助开发从概念走向日常。**Microsoft 365 Copilot** 则将 AI 嵌入 Word、Excel、Teams 等数亿用户每天使用的工具中。

这种"AI 无处不在"的策略使 Microsoft 同时在云计算、开发者工具和办公生产力三条战线上建立了 AI 壁垒。对 [Anthropic](/glossary/anthropic) 和 [Amazon](/glossary/amazon) 等竞争对手来说，Microsoft 的生态整合能力是最大的挑战。详细的 AI 代理发展趋势可参考我们的[博客分析](/blog/anthropic-cowork-claude-desktop-agent)。

## Microsoft 的 AI 布局

Microsoft 的 AI 体系围绕几个核心层展开：

- **模型层**：通过 OpenAI 合作获得 GPT 系列独家云端分发权，同时自研 Phi 系列小模型
- **基础设施层**：Azure 提供 GPU 集群、推理优化和模型托管服务，是全球第二大云服务商
- **应用层**：Copilot 品牌统一覆盖编程（GitHub）、办公（M365）、安全（Security Copilot）等场景
- **开发者工具**：VS Code 生态 + GitHub 平台，覆盖全球过亿开发者

## 相关术语

- **[Anthropic](/glossary/anthropic)**：[Claude](/glossary/claude) 的开发商，Microsoft 在 AI 安全领域的主要竞争对手之一
- **[Amazon](/glossary/amazon)**：通过 AWS Bedrock 提供多模型服务，与 Microsoft Azure 直接竞争
- **[Claude](/glossary/claude)**：Anthropic 开发的大语言模型系列，Azure 和 AWS 均提供托管服务
