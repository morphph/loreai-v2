# Amazon — AI 术语表

**Amazon** 是全球最大的科技公司之一，旗下 Amazon Web Services（AWS）是云计算市场的领导者。在 AI 领域，Amazon 既是基础设施提供商（通过 AWS 提供 GPU 算力、模型托管服务），也是 AI 应用的深度使用者——从电商推荐系统到 Alexa 语音助手，再到 Amazon Bedrock 模型服务平台。

## 为什么 Amazon 重要

AWS 占据全球云计算市场约 30% 的份额，是大量 AI 公司训练和部署模型的首选平台。[Anthropic](/glossary/anthropic) 与 Amazon 建立了深度合作关系，Amazon 对 Anthropic 进行了数十亿美元的投资，[Claude](/glossary/claude) 模型也通过 Amazon Bedrock 向企业客户提供服务。

对于 AI 开发者来说，AWS 提供了从数据存储（S3）、模型训练（SageMaker）到推理部署（Bedrock、Inferentia 芯片）的完整工具链。这使得 Amazon 成为 AI 生态系统中不可绕过的基础设施层。更多关于 Claude 生态的最新动态，可以参阅我们的[相关报道](/blog/anthropic-claude-memory-upgrades-importing)。

## Amazon 在 AI 领域如何运作

Amazon 的 AI 战略围绕三个层面展开：

- **基础设施层**：自研 Trainium 和 Inferentia 芯片，降低 AI 训练和推理成本，与 NVIDIA GPU 形成互补
- **平台层**：Amazon Bedrock 提供多模型托管，开发者可以通过统一 API 调用 Claude、Llama 等主流模型，无需自行管理基础设施
- **应用层**：Amazon Q 是面向企业的 AI 助手，覆盖代码生成、业务分析等场景；Alexa 则持续整合大语言模型能力

## 相关术语

- **[Anthropic](/glossary/anthropic)**：Amazon 重点投资的 AI 安全公司，Claude 模型的开发者
- **[Claude](/glossary/claude)**：Anthropic 开发的大语言模型，通过 Amazon Bedrock 提供企业级服务
- **[Google](/glossary/google)**：Amazon 在云计算和 AI 领域的主要竞争对手，运营 Google Cloud 和 Gemini 模型
