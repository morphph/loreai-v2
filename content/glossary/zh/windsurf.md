# Windsurf — AI 术语表

**Windsurf** 是 Codeium 公司推出的 AI 原生代码编辑器，基于 VS Code 开源内核构建，将大语言模型深度集成到编辑、导航和重构的全流程中。它的核心卖点是 Cascade——一个能感知整个项目上下文的 AI 代理，可以跨文件规划并执行多步骤编码任务。

## 为什么 Windsurf 值得关注

AI 辅助编程工具正在从"行级补全"向"[Agentic Coding](/glossary/agentic-coding)"演进。Windsurf 是这个趋势中的重要玩家，它试图在保留传统 IDE 编辑体验的同时，提供接近自主 Agent 的能力。对开发者来说，这意味着不需要在编辑器和终端之间切换——在一个界面里完成从理解代码到修改代码的完整循环。

Codeium 此前以免费代码补全插件积累了大量用户，Windsurf 是其从插件走向独立产品的关键一步。关于 AI 编程工具的最新竞争格局，可以参考我们的[相关报道](/blog/openai-computer-access-agents-lessons)。

## Windsurf 的工作原理

Windsurf 的技术架构围绕两个核心能力展开：

- **Cascade Agent**：项目级上下文感知的 AI 代理，能读取整个代码库结构，理解文件间依赖关系，在多个文件中协调执行编辑任务
- **Flows 模式**：介于自动补全和完全自主之间的交互方式，AI 跟随开发者的编辑意图实时提供建议，开发者保持最终控制权

Windsurf 支持接入多种大语言模型，用户可以根据任务复杂度选择不同的模型后端。编辑器本身兼容 VS Code 扩展生态，降低了迁移成本。

## 相关术语

- **[Agentic Coding](/glossary/agentic-coding)**：AI 代理自主规划并执行编码任务的开发范式，Windsurf 的 Cascade 是这一方向的实现
- **[ChatGPT](/glossary/chatgpt)**：OpenAI 的对话式 AI，部分开发者用它辅助编程，但缺乏 Windsurf 这样的 IDE 深度集成
- **[Claude Desktop](/glossary/claude-desktop)**：Anthropic 的桌面端 AI 助手，侧重通用任务而非代码编辑
