# Qwen 3.5 — AI 术语表

**Qwen 3.5**（通义千问 3.5）是阿里巴巴推出的新一代大语言模型系列。作为 Qwen 系列的最新迭代，Qwen 3.5 在推理、代码生成和多语言能力上实现了显著提升，同时提供从轻量级到超大规模的多种参数版本，覆盖从边缘部署到云端推理的完整场景。

## 为什么 Qwen 3.5 值得关注

Qwen 3.5 是目前开源大模型领域的重要竞争者。阿里巴巴以 Apache 2.0 协议开源了多个参数版本，开发者可以自由下载、微调和商用部署。这对不希望依赖闭源 API 的团队来说意义重大——你可以在自己的基础设施上运行一个能力接近前沿水平的模型，同时保持对数据和成本的完全控制。

Qwen 3.5 系列还引入了混合思维模式，模型可以在快速响应和深度推理之间动态切换，兼顾效率与准确性。这种 [agentic](/glossary/agentic) 能力让它在工具调用和多步骤任务中表现出色。

## Qwen 3.5 的技术特点

Qwen 3.5 基于 Transformer 架构，采用了多项工程优化：

- **多版本覆盖**：提供从数十亿到数千亿参数的多种规格，适配不同算力预算
- **混合推理**：支持"思考模式"和"快速模式"切换，按需分配计算资源
- **长上下文支持**：支持超长文本输入，适合文档分析和代码理解场景
- **多模态扩展**：系列中包含视觉语言模型变体，可处理图文混合任务

模型权重托管在 Hugging Face 和 ModelScope 上，配合 vLLM 等推理框架可快速部署。

## 相关术语

- **[Claude](/glossary/claude)**：[Anthropic](/glossary/anthropic) 推出的大语言模型系列，Qwen 3.5 的主要竞品之一
- **[Agentic](/glossary/agentic)**：AI 系统自主规划和执行任务的能力，Qwen 3.5 的混合推理模式对此有直接支持
- **[Anthropic](/glossary/anthropic)**：专注于 AI 安全的公司，其 Claude 模型与 Qwen 3.5 在多项基准上形成竞争
