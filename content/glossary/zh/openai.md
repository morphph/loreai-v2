# OpenAI — AI 术语表

**OpenAI** 是一家总部位于旧金山的人工智能公司，创建了 GPT 系列大语言模型和 ChatGPT 产品。公司成立于 2015 年，最初定位为非营利研究机构，后转型为"有限盈利"结构，目前是全球估值最高的 AI 公司之一。

## 为什么 OpenAI 重要

OpenAI 几乎以一己之力将大语言模型从学术研究推向了大众应用。2022 年底 ChatGPT 的发布引爆了全球对生成式 AI 的关注，直接推动了 [Anthropic](/glossary/anthropic)、[Google](/glossary/google) 等公司加速布局 AI 产品。

OpenAI 的 API 平台是目前开发者生态最成熟的 LLM 接口之一，大量 AI 应用和创业公司构建在其基础之上。其定价策略和模型迭代节奏在很大程度上定义了行业标准。关于 AI 公司间的竞争格局，可以参考我们的 [Anthropic Claude 记忆功能分析](/blog/anthropic-claude-memory-upgrades-importing)。

## OpenAI 的核心产品

OpenAI 的产品线围绕几个核心方向展开：

- **GPT 系列模型**：从 GPT-3 到 GPT-4o，持续提升多模态理解和推理能力
- **ChatGPT**：面向消费者的对话产品，拥有超过一亿周活跃用户
- **API 平台**：为开发者提供模型调用接口，支持 function calling、embeddings、图像生成等能力
- **DALL·E**：文本生成图像模型
- **Codex / GPT 编程能力**：驱动了 GitHub Copilot 等代码辅助工具的早期版本

OpenAI 也在积极探索 AI 智能体（agent）方向，推出了 Operator 等产品尝试。
